# vlukyanets/noctalia-plugins

A personal [Noctalia](https://noctalia.app) v4 plugin registry.

## Plugins

| Plugin | Description |
|---|---|
| [draind](draind/) | Show and control [draind](https://github.com/vlukyanets/draind) power management profiles from the menu bar |

## Adding this registry in Noctalia

Paste the repository URL in **Settings → Plugins → Add Source**:

```
https://github.com/vlukyanets/noctalia-plugins
```
